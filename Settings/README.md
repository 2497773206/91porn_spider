1.下载ffmpeg
```
https://www.johnvansickle.com/ffmpeg/old-releases/     ffmpeg下载地址
wget https://www.johnvansickle.com/ffmpeg/old-releases/ffmpeg-4.3.2-amd64-static.tar.xz
xz -d ffmpeg-4.3.2-amd64-static.tar.xz
tar -xvf ffmpeg-4.3.2-amd64-static.tar
cd ffmpeg-4.3.2-amd64-static
cp ffmpeg /home/centos/91porn_spider-main/Settings/             /home/centos/根据自己实际路径
```
然后把ffmpeg移动到Settings目录下
